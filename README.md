# proskell-web-side
### Use with python3.

## Before start 
- Add virtual env:
    - on windows: "py -3 -m venv venv"
- Then activate your venv:
    - on windows: "venv\Scripts\activate"
- Install flask within venv: "pip install Flask"
- Set flask env: set FLASK_APP=flaskr set FLASK_ENV=development flask run

OR
- pip install -e .
